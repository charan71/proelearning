Oracle DBA Course Content

ORACLE DBA TRAINING (Versions Oracle 9i, 10g, 11g)

BASICS:

  • Introduction to Database management systems and Database
  • Introduction to DBA and Tasks of an Oracle DBA
  • Basic UNIX and UNIX for Oracle DBA

DATABASE ARCHITECTURE:

  • Database Architecture
  • Control Files, Redo Log Files
  • Tablespaces and Datafiles, Segments, Extents, and Blocks
  • Oracle Instance Management and Oracle Memory Structures, Oracle Processes

ORACLE DATABASE STRUCTURE AND STORAGE:

  • Storage Structures
  • Tablespace Management (Permanent, Temporary, UNDO)
  • Data Files, Redo Log files, Archived log files, Temp files, spfile, pfile

USER AND OBJECT MANGEMENT, SECURITY:

  • Creating a Database User, Database User Accounts
  • Creating Profiles and Users, Authenticating Users
  • Privileges: System and Object
  • Assigning Quota to Users, Assigning Roles to Users
  • Managing Schema Objects (Tables, Indexes, Partitioned Tables and Indexes, Views, Sequences and Synonyms)

INSTALLATION:

  • Optimal Flexible Architecture (OFA)
  • Oracle Universal Installer
  • Prerequisite Checks
  • Installation of Oracle Database Software

CREATING ORACLE DATABASE:

  • Database Configuration Assistant (DBCA) Overview
  • Creating a Database (Manual and DBCA)
  • Passwords and Storage
  • File Locations and Backup Recovery
  • Content and Initialization Parameters
  • Other Actions with DBCA

DATABASE MANGEMENT:

  • Starting and Stopping Database
  • Initialization Parameter Files
  • Viewing the Alert Log
  • Net Services

BACKUP AND RECOVERY:

  • Backup and Recovery Concepts, Backup and Recovery Issues
  • Physical and Logical Backups
  • Types of Physical Backups (HOT and COLD) --Manual and RMAN Backups
  • Recovery Scenarios using RMAN and Manual backups
  • Database Instance Cloning

MONITORING ORACLE:

  • Dictionary and v$views
  • Table and Index monitoring
  • Instance or Database monitoring
  • Oracle Environment and work load monitoring
  • Space Management
  • Monitoring alert log, common database errors and resolution

PERFORMANCE MONITORING and TUNING:

  • Database Statistics, Explain plan, indexing
  • AWR reports, ADDM reports, ASH reports and analyzing the reports
  • SQL Tracing and TKPROF
  • Memory Advisors, Segment Advisors
  • SQL Tuning and Access Advisors
  • Wait Events in Oracle, Locking, Parallelism
  • Monitoring performance on the Server - UNIX
  • How to monitor and react to performance issues

ORACLE DBA UTILITIES:

  • Export/Import, Datapump
  • SQL *Loader

DATABASE MAINTENANCE:

  • Database Version Upgrade
  • Patching Database
  • Migrating Data and Databases (includes TTS)
  • Re-organizing tables, indexes, Flashback

HIGH AVAILABLITY (ASM, RAC and DATAGUARD)

ASM (Automatic Storage Management):

  • Introduction to ASM
  • ASM Instance Architecture
  • Installation and Configuration of ASM
  • Oracle Managed Files (OMF)

RAC (Real Application Clustering):

DATAGUARD (Standby Database or DR) Setup:

  • Introduction to Dataguard
  • Setup of Dataguard or Standby database on Single Instance
