About The Course

Introduction to UNIX Course. In this course, students learn the fundamentals of the UNIX operating system, such as file editing, data retrieval, executing processes and creating directories.

An introduction to shell scripts may be covered depending on the program.

This hands on course provides training on standard UNIX/Linux commands and utilities used for day to day tasks including file manipulation, program execution and control, and effective use of the shell and desktop environments.

The course presents the concepts necessary to understand the way UNIX works as well as the system's most commonly used commands. Data manipulation utilities and shell syntax for synthesizing command pipelines are emphasized.

Bourne shell, Bash shell and Korn shell programming techniques are introduced so students will be able to read and modify existing shell scripts as well as create their own.

Desktop environments are also introduced from a user's perspective, including common window managers, Open Office utilities and an introduction to configuration tools. Comprehensive hands on exercises are integrated throughout to reinforce learning and develop real competency.

Course Overview

1. Introduction

  • The UNIX OS
  • Unix architecture
  • Linux & GNU
  • Brief History
  • Logging into Unix
  • Basic cat and ls command
  • Help page (man)

2. General Purpose Utilities

  • Cal and date command
  • Echo, printf, bc command
  • Passwd, who, uname, tty

3. File System

  • Files and Inodes
  • Unix file system
  • Listing files and Directories and options
  • Creating & deleting files and directories
  • Pwd, cd commands
  • Move and copy files and directories
  • Abs or relative path names

4. VI Editor

  • Vi basics
  • Saving text and editing
  • Navigation
  • Undo, Search, Repeat
  • Substitution

5. Working with files

  • Cat, cp, rm, mv, more
  • File, wc, diff, dos2unix

6. File Attributes

  • File permissions
  • Directory permissions
  • File ownership
  • Changing file and directory permissions
  • Changing file ownership

7. Shell

  • Using wild cards in shell
  • Pipes
  • Output redirection
  • Command substitution
  • Re-direction: inp, out, err
  • Shell variables or environment variables

8. Process

  • Process basics
  • Ps command
  • System processes
  • Jobs
  • Background jobs
  • Job control
  • At or cron
  • Killing process

9. File attributes

  • Inodes
  • Hard links and soft links
  • Creating hard and soft links
  • Umask
  • Find command to find files

10. Filter commands

  • Head and tail
  • Grep and cut command
  • Sort command

11. Database

  • How to connect to a database
  • Fetch query result to a variable

12. System

  • Root user privileges
  • User management (User add and delete)
  • /etc/passwd and /etc/shadow files
  • Tar and zip commands
