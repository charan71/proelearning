Prerequisites

Before learning IoT, you need to know the following

  • Operating Linux Systems (Debian Linux)
  • Network protocols (OSI and TCP/IP) model.
  • Electronic circuits.
  • Programming: Python/C++/Ruby on Rails.

About The Course

What is RPA?

Robotic Process Automation is a platform to reduce human efforts by deploying a Robot or Software to perfrom specific task which is repeatedly performed. RPA tools will interact with business applications as a human user would do.

  • ASIC in Latest Technology
  • Basic Micro Controller Concept
  • Embedded System Application

  • Digital Read, Digital Write

  • Bluetooth Communication

Module 4

Module 5

  • Introduction to Programming in Embedded
  • Introduction to Relay Driver IC
  • Introduction to ADC
  • Introduction to WIFI

Practicals

  • PRACTICAL 1: Blinking of LED

Projects

  • PROJECT 6: Real time monitoring of sensors on a particular IP address/server
