About The Course

This course specially target to developers, administrator, cloud engineers, network administrators.

OpenStack Certification Training makes you an expert by mastering concepts like Object Storage, Networking, Compute, Nova Architecture, Usage and use cases of swift operations, Orchestration, etc.

Course Overview

1. Introduction to Cloud Computing

  • Virtualization Overview
  • Software as a service
  • Platform as a service
  • Infrastructure as a Service
  • Openstack – History and Overview

2. OpenStack Architecture DeepDive

  • Components Overview
  • Architecture
  • Keystone
  • Nova
  • Glance
  • Neutron
  • Horizon
  • Heat
  • Cinder
  • Swift
  • VM creation flow

3. Deploying OpenStack – Step by Step through CLI

  • Getting the pre-requisites right
  • Setting up MySQL, RabbitMQ
  • Setting up Keystone
  • Setting up Glance
  • Setting up Nova
  • Setting up Neutron
  • Setting up Dashboard
  • Setting up Cinder

4. Deploying OpenStack – Step by Step through CLI

  • Understanding Cloud Images and instances
  • Registering and uploading an image
  • Creating keypairs, Security Groups
  • Creating and Managing Instances from CLI and Dashboard
  • Assing floating point IP to VM
  • Log Files, Troubleshooting, Common Errors

5. Puppet

  • Introduction to Puppet
  • How to install puppet master and agents
  • Creating resources and classes
  • Installing openstack using puppet

6. Openstack High Availability set up

  • High availability architecture
  • High availability components
  • Advantage of using high availability
