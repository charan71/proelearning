About The Course

Simplilearn’s online CCSP training course will help you gain expertise in cloud security architecture, design, applications, and operations. CCSP by ISC2 is a globally acknowledged certification which represents the highest standard for Cloud security.

This course provides step by step guidance and easy-to-follow detailed explanation on every facet of CCSP.

Architectural Concepts and Design Requirements

1.1 Understand Cloud Computing Concepts

  • Cloud Computing Definitions (ISO/IEC 17788)
  • Cloud Computing Roles (i.e., Cloud Service Customer, Cloud Service Provider,
    and Cloud Service Partner)
  • Key Cloud Computing Characteristics (e.g., on-demand self-service, broad network access,
    multi-tenancy, rapid elasticity and scalability, resource pooling, measured service)
