About The Course

The goal of this course is to introduce the students to the basics of distributed application development. We will introduce the students to Web Service development and .NET remoting.

Technologies covered include the Common Language Runtime (CLR), .NET framework classes, C#, ASP.NET, and ADO.NET.

We will also cover service oriented architecture, design, performance, security, content managements systems and deployment issues encountered in building multi-tier distributed applications.
